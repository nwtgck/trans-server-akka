Command line usage

Send with wget

wget -q -O - {{location.protocol}}//{{location.hostname}}{{(location.port == 80? '': ':'+location.port)}}/ --post-file=./test.txt
  • "./text.txt" is a file you want send
  • "-q": Disable progress bar
  • "-O -": outputs /dev/stdout

Get with wget

wget {{location.protocol}}//{{location.hostname}}{{(location.port == 80? '': ':'+location.port)}}/ab2
  • "ab2" is a file ID, which is generated after sending
